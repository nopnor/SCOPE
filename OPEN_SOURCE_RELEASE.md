# SCOPE Open-Source Export

This directory is a sanitized source export of SCOPE.

Included:

- Python package under `src/scope`
- Codex workflow skills under `.codex`
- configuration templates under `configs`
- public docs under `docs`
- tests under `tests`
- Gen-Arena materialization and evaluation utilities
- MIT license metadata

Excluded:

- generated images and run artifacts
- benchmark materialized case packets and evaluation outputs
- third-party benchmark adapters and evaluator-compatibility notes
- local temporary directories and caches
- paper drafts and review notes
- internal one-off experiment and baseline scripts
- API keys and private environment files

Before publishing, verify benchmark data redistribution terms separately.
